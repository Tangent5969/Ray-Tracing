# Ray Tracing

Simple ray tracing rendering using C++ and OpenGL.

Work in progress.

Heavily inspired by [Sebastian Lague](https://youtu.be/Qz0KTGYJtUk).

## Controls
 
* W - Forwards  
* A - Left  
* S - Back  
* D - Right
* Space - Up
* Control - Down
* Shift - Sprint
* Mouse & Arrows - Pan Camera
* Escape - Exit Program
