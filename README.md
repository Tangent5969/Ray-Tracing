# Ray Tracing

Simple ray traced rendering using C++ and OpenGL.

Work in progress.

Inspired by [Sebastian Lague](https://youtu.be/Qz0KTGYJtUk) and 
[Ray Tracing In One Weekend](https://raytracing.github.io/).

## Controls
 
* W - Forwards  
* A - Left  
* S - Back  
* D - Right
* Space - Up
* Control - Down
* Shift - Sprint
* Mouse & Arrows - Pan Camera
* Escape - Exit Program
